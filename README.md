# Northeastern Class Schedule Maker

### Aidan Kelly

Generates optimal class schedules for Northeastern students based on the classes they choose.

**Frontend:** Typescript, React, Vite, and Tailwind  
&nbsp;&nbsp;&nbsp;&nbsp; _Hosted on Cloudflare._

**Backend:** Python, TanStack Query, and FastAPI  
&nbsp;&nbsp;&nbsp;&nbsp; _Hosted on Render._

Class data pulled form Northeastern's Sandbox club's SearchNEU public class API.

Use at https://nu-schedule-maker.aidanjkelly.com/.
&nbsp;&nbsp;&nbsp;&nbsp; _The backend may take up to a minute to become active due to Render's free plan restrictions._
